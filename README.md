Project Name: Cars and Coffee

Description
Welcome to Cars and Coffee — a community of car enthusiasts sharing their passion over early mornings and coffee.  
This website includes sections for car meets, a coffee menu, individual car showcase page, and a sign‑up form.  

GitHub Link
[Cars and Coffee Repository](https://github.com/ioniq25/cars_and_coffee.git)

Technologies Used
- HTML  
- CSS  
- Bootstrap  
- JavaScript
